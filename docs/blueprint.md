# **App Name**: WiraData

## Core Features:

- Data Input: Form to input demographic data including NIK, No. KK, full name, phone number, address, business type, business location and coordinator.
- Data Storage: Store the entrepreneur data, creating and maintaining a comprehensive database.
- Search and Filter: Enable users to search and filter entrepreneur data based on different criteria (e.g., name, location, business type).
- Data Export: Functionality to export data in common formats (e.g., CSV, Excel) for further analysis or reporting.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke a sense of reliability and sophistication.
- Background color: Very light grey (#F0F0F0) for a clean and professional look.
- Accent color: Complementary yellow-gold (#FFC857) to highlight key actions and data points.
- Body font: 'Inter' for clear, neutral readability of data fields and descriptions.
- Headline font: 'Space Grotesk' for headings and titles to maintain a techy but readable style.
- Use clear, professional icons to represent data types and actions (e.g., user icon for name, location pin for address).
- Maintain a clean, tabular layout for data input and display for clarity.
